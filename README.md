# TickMe – Task & Calendar

TickMe is a SwiftUI productivity app for managing tasks, schedules, reminders, and calendar planning.

## Features
- Task creation and management
- Schedule and calendar planning
- Reminder support
- Light and dark mode
- Privacy permissions for Calendar, Notifications, and Photos

## Tech Stack
- SwiftUI
- Xcode
- iOS
- UserNotifications
- EventKit
